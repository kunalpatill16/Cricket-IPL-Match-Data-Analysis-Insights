# Cricket-IPL-Match-Data-Analysis-Insights 
An interactive IPL Cricket Analytics Dashboard developed using Python, Streamlit, Pandas, and Plotly. The project provides player analysis, team performance insights, venue statistics, player comparison, and match prediction through dynamic visualizations and data analytics.

## Dataset Information

⚠️ Note:

The original IPL dataset (`IPL.csv`) used in this project was larger than GitHub's maximum file size limit (100 MB). Therefore, the dataset has not been uploaded to this repository.

To run this project successfully, please download the dataset from Kaggle and the link provided below and place it in the project root directory.

Dataset Link:
[https://www.kaggle.com/code/arbazkhan971/indian-premier-league-analysis-2008-2025/notebook]

After downloading, ensure that the dataset file is named:

IPL.csv

and is placed in the same directory as `app.py`.


#Runing command :-
streamlit run app.py
